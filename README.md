# Калькулятор

## Задание

Напишите модуль с функциями для базовых математических операций.

### Требования

Файл с решением: `src/calculator.js`

Экспортируйте 4 функции:

1. `add(a, b)` — возвращает сумму a и b
2. `subtract(a, b)` — возвращает разность a и b (a - b)
3. `multiply(a, b)` — возвращает произведение a и b
4. `divide(a, b)` — возвращает частное a / b

### Обработка ошибок

- При делении на ноль выбрасывайте ошибку с текстом: `'Деление на ноль'`
- Все аргументы должны быть числами. Если передан не-число — выбрасывайте ошибку: `'Аргумент должен быть числом'`

### Примеры

```javascript
const { add, subtract, multiply, divide } = require('./src/calculator');

add(2, 3)        // 5
subtract(10, 4)  // 6
multiply(3, 5)   // 15
divide(10, 2)    // 5
divide(5, 0)     // Error: Деление на ноль
